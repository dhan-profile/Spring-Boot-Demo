# Spring Boot Hello World Example
Get started with the Spring Boot application, a hello world example.


## 1. How to start
```bash
$ git clone [https://github.com/dhan-profile/Spring-Boot-Demo.git](https://github.com/dhan-profile/Spring-Boot-Demo.git)

$ cd Spring-Boot-Demo

# Tomcat started at 8080
$ mvn spring-boot:run

# test
curl localhost:8080

```
